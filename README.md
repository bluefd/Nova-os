# Nova OS — Personal AI Workspace

Nova OS is an advanced web-based developer portfolio and conceptual personal AI workspace designed in a visionOS Glassmorphism style.

## Technologies Used
- **React 19 + TypeScript**
- **Tailwind CSS 4.0**
- **Framer Motion + GSAP** (for premium liquid motion)
- **Node.js + Express** (Backend proxy for secure model generation)
- **Gemini 3.5 Flash** (Cognitive assistant integration)
- **JSZip** (Dynamic browser code compiler)

## Getting Started

1. Clone or extract the folder.
2. Run `npm install` to install dependencies.
3. Add your `GEMINI_API_KEY` in your `.env` file.
4. Run `npm run dev` to boot up the environment.
5. Visit `http://localhost:3000`.
